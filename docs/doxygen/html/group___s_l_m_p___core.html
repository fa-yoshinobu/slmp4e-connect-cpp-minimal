Subcommand selection for device access.

Determines whether to use modern iQ-R extended subcommands or legacy Q/L subcommands.

Enumerator
iQR 

Use subcommands 0x0002/0x0003 (iQ-R series extensions). Supports larger address ranges.

Legacy 

Use subcommands 0x0000/0x0001 (Legacy Q/L series).
