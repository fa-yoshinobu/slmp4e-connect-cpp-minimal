## 2. Deep Dive into Failures

### PLC Error 0xC05B (Mixed Block Write)
The test attempted to write a mixed payload of words (D) and bits (M) in a single `writeBlock` request.
- **Request**: `50 00 00 FF FF 03 00 ...` (3E Header)
- **End Code**: `0xC05B` (`direct_g_hg_path_rejected`)
- **Conclusion**: This is an expected behavior of the iQ-R series hardware when processing mixed device types in one SLMP block command, confirming the library's error handling is correct.

### Header/State Mismatches (Target & Monitor)
These failures are likely due to the test sequence expectations not being fully aligned with the 3E frame's header structure or simulator-like behaviors on real hardware.
